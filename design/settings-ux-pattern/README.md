# Settings Screen UX Design — Commitlabs

## Overview
This folder contains UX design documentation for the Commitments Settings screen.  
It defines how users manage preferences, notifications, privacy, and account controls in a structured and accessible way.

This work is UI/UX only (no frontend implementation).

## Scope
The settings system covers:

- User preferences (e.g. currency display, display options)
- Notification controls
- Privacy settings
- Account actions (e.g. logout)


## What’s included

- Information Architecture (IA)
- UX behavior and interaction rules
- Mobile and desktop layout specifications
- Screen designs (Figma exports or images)
