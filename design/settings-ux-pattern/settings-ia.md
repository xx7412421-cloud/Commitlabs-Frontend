# Settings Information Architecture (IA)

## Purpose
This document defines how settings are structured and grouped for clarity, usability, and scalability.


## Settings Structure

### 1. Profile
- Full name 
- Email address
- Avatar
= Role badge


### 2. Currency & display
- Default currency
- Number format 
- Date format 
- Compact numbers 
- Dark mode


### 3. Notifications
- Maturity alerts 
- At-risk changes 
- Violation alerts 
= Export completed 
- Delivery channel

### 4. Privacy & security
- Two-factor auth 
- Analytics sharing 
= Public share links 
= Active sessions

## Navigation Pattern
- Each section is grouped visually
- Sections are separated with clear spacing
- On mobile, sections stack vertically
- On desktop, sections are grouped with clear hierarchy blocks
